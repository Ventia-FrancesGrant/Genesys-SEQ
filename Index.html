⚠️ Missing Information Protocol

Use if any checklist item above is missing.

  3. CREATE the event in QFM.
  4. ASSIGN to the FM.

Use if all checklist items are present.

  1. CREATE the event in the system immediately.
  2. CONTACT the FM for assignment.
